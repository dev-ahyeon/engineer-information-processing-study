# Section 057 | 연계 메커니즘 (C)

### 1. 연계 매커니즘
- 방식
  1. 직접 연계 방식 
  - 정의 : 중간 매개체 없이 송ㆍ수신 시스템이 직접 연계하는 방식
  - 종류 : DB link, API/Open API, DB Connection, JDBC 등
  2. 간접 연계 방식 
  - 종류 : 송ㆍ수신 시스템 사이에 중간 매개체를 두어 연계하는 방식
  - 종류 : 연계 솔루션, ESB, 소켓(Socket), 웹 서비스(Web Service) 등
